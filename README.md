# Mechanical Engineering Portfolio Website

This folder contains a static portfolio website for Nabidul Islam.

## Pages
- `index.html` — About landing page
- `resume.html` — Full-screen resume page with print/save button
- `projects.html` — Project index
- `nasa-lspace-thermal-control.html`
- `garage-door-mechanical-system.html`
- `six-dof-robotic-arm.html`
- `lightweight-hybrid-bicycle.html`
- `urban-climate-analysis.html`
- `gripwell-assistive-device.html`

## How to use
Open `index.html` in a browser. The site is fully static and can be uploaded to GitHub Pages, Netlify, Vercel, or any basic web host.

## Assets
Project visuals were extracted from the uploaded project PDFs when images were available. GripWell uses a custom vector illustration because no project image was uploaded.
